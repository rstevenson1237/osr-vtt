11

Token status ring (group / selection / ownership)

R21 · WI-24

Every map token gets an outer ring the engine draws from live state: black = no group · group color = in a group · white = selected or owned by you. Precedence: white (selected/owned) > group color > black.

Ring states
  • Selection is transient; ownership is persistent — both white. A grouped token I select goes white, then back to group color when deselected.
  • Separate from a generated disc's own baked art ring (R7.1) — this is a render-time overlay driven by groupId / ownerSeatId / selection.
  • Optional later: if selected vs. owned should differ, split as solid-white (owned) vs. white + subtle glow (selected). Not in v1 unless you want it.
